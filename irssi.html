Configuration

아직도 cp949 인코딩을 사용하고 있는 HanIRC 등을 위해 irssi의 recode 기능을 설정할 필요가 있음. (참고: UTF-8 터미널 환경에서 irssi로 hanirc 이용하기)

irssi 실행 후, 하단 입력 칸에 한줄 씩 다음 내용을 입력할 것. 마지막 save 명령은 이상의 내용을 설정 파일에 저장시킨다. 


/set recode_autodetect_utf8 OFF

/set recode_fallback cp949

/set recode ON

/set recode_out_default_charset cp949

/set recode_transliterate ON

/save

irssi를 종료하고, 설정 파일(~/.irssi/config)에 위의 내용이 잘 저장되었는지 확인하고 필요하다면 기본 nick 등도 설정해준다.
