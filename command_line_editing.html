Key

C-a		라인의 처음으로

C-e		라인의 끝으로

M-f		한 단어 앞으로

M-b		한 단어 뒤로

C-k		커서 위치에서 라인 끝까지 삭제

C-u		커서 위치에서 라인 처음까지 삭제

M-d		커서 위치에서 현재 단어의 끝까지 삭제

M-<Backspace>	커서 위치에서 현재 단어의 처음까지 삭제

C-w		커서 위치에서 이전 공백까지 삭제

C-y		최근에 삭제한 내용 붙여넣기

C-_		Undo

C-r		Reverse search

Reverse search

$
(reverse-i-search)`':						# C-r
(reverse-i-search)`s': sudo vi sshd_config			# s
(reverse-i-search)`su': sudo vi sshd_config			# u
(reverse-i-search)`sud': sudo vi sshd_config			# d

(reverse-i-search)`sud': sudo find / -name "sshd_config"	# C-r
$ sudo find / -name "sshd_config"				# ESC
