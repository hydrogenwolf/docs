dig

기본 사용. (Default 네임서버는 resolv.conf에서 참조. Default 레코드는 A) 

$ dig google.com

네임 서버 지정. 

$ dig @8.8.8.8 google.com

대상 레코드를 MX로 지정. 

$ dig google.com mx

네임 서버 지정하고 대상 레코드는 NS로 지정. 

$ dig @8.8.8.8 google.com ns

SOA 

$ dig google.com soa

ANY 

$ dig google.com any

Reverse Lookup 

$ dig -x 66.249.89.99

ANSWER 섹션만 표시. 

$ dig google.com +noall +answer

; <<>> DiG 9.5.0-P2.1 <<>> google.com +noall +answer
;; global options:  printcmd
google.com.             9       IN      A       66.249.89.99
google.com.             9       IN      A       66.249.89.104

순수하게 결과만 표시. 

$ dig google.com +short
66.249.89.104
66.249.89.99

Lookup 과정 표시. 

$ dig google.com +trace
